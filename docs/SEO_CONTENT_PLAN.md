# WashDog SEO Content Plan

Keyword strategy and content roadmap for WashDog's organic growth in Santiago, Chile.

**Primary market:** Ñuñoa
**Secondary markets:** Providencia · Las Condes · La Reina
**Language:** Spanish (Chile) — natural, conversational, never formal

---

## Strategic Overview

WashDog's SEO opportunity is concentrated in **high-intent, local keywords** — people searching for dog grooming services near their home in Santiago. The goal is to rank in the top 3 for every service + location combination in Ñuñoa, then expand to neighboring comunas.

**Three content layers:**
1. **Service pages** — transactional intent, direct booking conversion
2. **Blog posts** — informational intent, brand authority, long-tail traffic
3. **Location expansion pages** — same services, different comunas

## Technical SEO Requirements

Every page (service + blog) must include:

### Schema Markup

```json
// Service pages → LocalBusiness + Service
{
  "@type": "LocalBusiness",
  "name": "WashDog",
  "address": { "addressLocality": "Ñuñoa", "addressCountry": "CL" },
  "geo": { "latitude": -33.456, "longitude": -70.612 },
  "priceRange": "$$",
  "telephone": "+56 9 XXXX XXXX"
}
```

```json
// Blog posts → Article + BreadcrumbList
{
  "@type": "Article",
  "headline": "Cómo bañar a tu perro en casa",
  "datePublished": "2026-03-01",
  "author": { "@type": "Organization", "name": "WashDog" }
}
```

### Meta Requirements

| Element | Requirement |
|---|---|
| `<title>` | Keyword + brand + city, 50–60 chars |
| `<meta description>` | Keyword + CTA, 140–155 chars |
| `<h1>` | One per page, includes primary keyword |
| `<h2>` / `<h3>` | Related keywords, logical hierarchy |
| Internal links | Each blog post → service page (2+ links) |
| Image alt text | Descriptive, includes keyword when natural |
| Canonical URL | Self-referencing, no trailing slash issues |
